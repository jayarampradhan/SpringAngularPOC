**AngularSpringPoc**
======================

**Intro**
=====================
A Sample Project implementation using Spring, Thymleaf and Angular JS.


**Steps To Run**
=====================
Run As Java Application:
<pre>StartApp</pre>

Navigate to : <a href="http://127.0.0.1:8181/uim/springAngular">Local Host</a>


**Technology Stack**
====================
- jdk 1.8
- Spring 4.14
- Spring Boot 1.2.1
- Angular
- Thymleaf
